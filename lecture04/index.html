Character

A char type is a primitive datatype that represents a single character.

Character

A char type is a primitive datatype that represents a single character.

Unicode is of this format: \uXXYY where XX is the first hex set and YY is the second hex set.

Java (like C/C++) supports escape sequences which begins with '\' and followed by a special character or a unicode sequence.


Character casting is similar to other casting constructs. "Precision" will be lost -- in other words only the first 16 bits are retained.

To do various testing the state of a character, a Character class is used. For example, to test if a character is uppercase: Character.isUpperCase(c) will return a true if c is an uppercase character. Likewise for conversion, we use Character.toUpper(c).


Java Strings

A string is a sequence of characters.

  String message = "java is fun!";

You can also manipulate a String using its member methods.


Java provides concatenation operator (+) that you can use directly on a string. For example: 

  String message = "java is fun!";
  String m2 = message + " way too much fun!";

When concatenating a number (integer, float, double, etc) with a string that number is converted to a string.

  String message = "a number is " + 2;

Similar to Character class, you can manipulate a String object. 

  String k = "number".toUpperCase();

To read a string from the console, we have two options:

  1. To read one word at a time
  2. To read the entire line at once
   Scanner s = new Scanner(System.in);
   String k = s.next();
   String l = s.nextLine();

To compare strings, you must use the string's compare methods. Note that you cannot compare using the "==" operator.


You can also get part of a string with the substring(...) method

String message = "Welcome to Java";
String message = message.substring(0, 11) + "HTML";

If you have a number that is encoded as a String, you can use it respective Class to do the conversion.

   int k = Integer.parseInt("1" + "2");
   double j = Double.parseDouble("1.0" + "2");
